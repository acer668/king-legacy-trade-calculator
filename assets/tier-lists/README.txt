Tier list media folders.

The website loads image paths from values/tier-lists.json.
By default, each combo points to a file named tier-list.png in its matching folder.

Example:
assets/tier-lists/PvP/fruits/tier-list.png

If your image is named differently, edit values/tier-lists.json and set that combo to the exact path.

Accessories folders were added under both PvP and PvE. Default filename: tier-list.png
