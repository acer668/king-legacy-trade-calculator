Edit these JSON files to change website values without digging through script.js.

items.json: main Fin values used by the trade calculator.
robux-values.json: Robux prices used by the Robux Per Fin calculator. The website matches names from this file to items.json to find Fin values.
drop-values.json: drop names and original drop chances used by the Drop Boost Helper.

Keep valid JSON formatting: quotes around text, commas between entries, and no trailing commas.
